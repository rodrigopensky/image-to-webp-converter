# 🖼️ Image to WebP Converter

A lightweight and fast **Image to WebP Converter** built using pure JavaScript.

Convert images directly in your browser without uploading them to any server.

This ensures **privacy, speed, and security**.

---

## 🚀 Features

✅ Drag & Drop images  
✅ Multiple image upload  
✅ Instant WebP conversion  
✅ Download converted images  
✅ Fully client-side  
✅ Responsive layout

## ⚙️ How It Works

1. Upload or drag images
2. The browser loads the image
3. A canvas converts it to WebP
4. The result becomes downloadable instantly

All processing happens **inside your browser**.
